Just Animate - Easily animate text or images in the WordPress classic editor.


== Description ==

Just Animate is a simple WordPress plugin for animate text or images in your posts, easily create beautiful posts with animations.

How to use?
*   Select the text block or images you want to Animate with the cursor.
*   Click the Just Animate button at the top of the post editor.
*   Select animation type, speed and delay.
*   View the post and be amazed of the block animation as you scroll down the post.

Animation options:
*   Choose animation type (fade, zoom, slide and more).
*   Choose animation speed (slow, normal, fast, very fast).
*   Choose animation delay (no delay, 0.5 seconds, 1 seconds, 1.5 seconds).


== Installation ==

1. Unzip and upload the "just-Animate" folder to your '/wp-content/plugins/' directory
2. Activate the plugin in the 'Plugins' menu in WordPress.
3. Start animate text or images!


== Frequently Asked Questions ==

Unfortunately this plugin doesn't support Elementor.
